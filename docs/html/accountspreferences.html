功能

自动生成收费号

Asks: ___

默认：不自动生成收费号

值：

  • 默认：不自动生成收费号
  • 自动生成 <年>-0001 格式收费号
  • 自动生成 <分馆代码>yyyymm0001 格式收费号
  • 自动生成 1，2，3 格式收费号

描述：

  • 基于审计原因，每笔交易（如支付、注销、退款等）可能都需要一个唯一标识号，系统允许您自己选择唯一标识号的格式。

    重要

    需要为每一种 收费类型 启用自动生成。

启用销售点

Asks: ___ 销售点功能可以匿名与收费系统交易。

默认：关闭

值：

  • 关闭
  • 启用

描述：

  • 此参数控制 销售点模块。

重要

如果您启用此系统配置，要确保也启用了 使用收款机。

使用收款机

Asks: ___ 收款机配合款及系统可以追踪支付。

默认：不使用

值：

  • 不使用
  • 使用

描述：

  • 此参数启用管理模块的 收款机。

规则

自动对账

Asks: ___ 每次交易双方交易时自动对余额核对。

默认：关闭

值：

  • 启用
  • 关闭

描述：

  • 此参数控制是否启用自动扣款功能。

付款自动弹出窗口

Asks: ___ 付款时自动显示打印票据窗口。

默认：不显示

值：

  • 启用
  • 不显示

描述：

  • 如果激活该功能，当读者 :ref:`付款<pay-and-writeoff-fines-label>`时，不需要点击“打印”按钮而自动弹出打印窗口。

付款凑整

Asks: ___ 付款时对于金额会四舍五入。

默认：不显示

值：

  • 不显示
  • 启用

描述：

  • 启用词参数付款时会四舍五入到分。
