概述

Koha 简介

Koha 是第一款自由开源图书馆集成系统（ILS：Intergrated Library System）。该系统由来自世界各地的各类型图书馆、志愿者以及相关支持公司共同开发完成。

想要了解更多 Koha 相关信息可访问 Koha 官方站点：http://koha-community.org

Koha 推荐

所有系统推荐参数可参考 Koha 官方维基的开发者文档： http://wiki.koha-community.org

如果您使用 Koha 馆员端，那么强烈推荐使用最新版的 Mozilla Firefox 或者 Google Chrome 浏览器。Koha 社区不支持馆员端使用 Internet Explorer 浏览器。如果您所在组织机构使用微软产品，可以使用 Edge 浏览器替代 Internet Explorer。

使用本手册

本手册在不断完善中，如果您有编辑的建议可以通过 GitLab 或者 koha-docs 邮件列表 给 Koha 文档小组提交一个合并请求。

如果如果你看到有错误或者其它问题，请在 bugzilla 的文档模块提交问题报告。你也可以发送邮件到 koha-docs 邮件列表。

手册是按照 Koha 的模块组织的。在每个部分的开始都会有“在这里”的提示。这告诉您在 Koha 系统中如何找到对应内容。

例如：在这里： 更多 > 管理 > 全局系统参数

这会告诉你所要找的菜单选项在 Koha 馆员端顶部的哪里。

手册中遍布可直达手册其它部分的链接和展示应看到屏幕现实内容的截屏。

参与修订本手册

本手册是一个不断完善的文档，欢迎您参与编辑。

Koha 手册 由文档小组管理，但是这并不意味着我们其他人不能参与完善手册。

手册通过 Git 存储在 https://gitlab.com/koha-community/koha-manual （查看概要 在这里）

参与手册做贡献的说明在 Koha 维基 Koha wiki https://wiki.koha-community.org/wiki/Editing_the_Koha_Manual
